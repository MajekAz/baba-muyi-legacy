## Definition Of Done For Every Milestone

Each roadmap row inherits this definition of done in addition to its row-specific acceptance criteria, tests, security checks, documentation updates, suggested commit message, and deployment requirement.

- Requirements implemented or explicitly deferred.
- Lint, typecheck, build pass.
- Relevant automated tests pass.
- Manual acceptance notes recorded.
- Security and privacy impact reviewed.
- Documentation updated.
- Rollback plan known.
