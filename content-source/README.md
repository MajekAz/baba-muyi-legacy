# Content Source

Place approved Baba Muyi Legacy source material in this directory before running an import.

Recommended structure:

```text
content-source/
  approved-content.json
  images/
  documents/
  transcripts/
  subtitles/
```

Every content item should include:

- Stable ID
- Slug
- Title
- Date or transparent date label
- Privacy level
- Verification status
- Source reference
- Copyright owner and permission status
- Publication status

Do not place unapproved private family material here unless it is clearly marked private and not ready to publish.
